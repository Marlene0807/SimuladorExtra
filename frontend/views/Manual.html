Manual del Simulador

En esta sección encontrarás el manual completo del simulador de subestaciones eléctricas. Aquí se detallan los pasos necesarios para utilizar la herramienta, así como las funciones disponibles y consejos para un uso efectivo.

El simulador está diseñado para ayudar en el aprendizaje sobre subestaciones eléctricas. En esta sección, podrás familiarizarte con la interfaz y las opciones disponibles.

Uso del Simulador

Una vez configurado, puedes empezar a simular diferentes escenarios. Usa las herramientas disponibles para crear y modificar diagramas unifilares y analizar los resultados.

1. Manual: Encontrarás algunos pasos y tips a seguir para hacer buen uso del sitio, aclararas algunas dudas y podrás leer el manual.

2. Simulación: En la sección "Lienzo Elementos", puedes seleccionar imágenes de componentes y arrastrarlas al lienzo de simulación. También puedes conectarlos y guardar el estado actual, así como cargar un estado previamente guardado.

3. Contacto: Proporciona datos en los cuales puedes contactar con nosotros los desarrolladores del sitio para dudas, aclaraciones, o mejoras.

Ejemplos y Prácticas

En esta sección, se incluyen ejemplos prácticos para ayudarte a comprender cómo utilizar el simulador en diferentes situaciones.

funciones

● Eliminar Componentes: Opción para eliminar imágenes seleccionadas o todos los componentes del lienzo.
